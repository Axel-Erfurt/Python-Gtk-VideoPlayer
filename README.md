# Python-Gtk-VideoPlayer

## Requirenments
- python3
- python3-gi
- yt-dlp

## Usage

``python3 Gtk3WebViewPlayer.py``

or 

``python3 Gtk4VideoPlayer.py``

**You can play**

- local files
- web files
- youtube links
